# FontManager - 专业字体管理工具

## 简介

FontManager 是一款基于 PyQt6 的专业 Windows 字体管理工具，提供字体浏览、预览、比较、搜索、分类、导出等功能。

## 功能特性

### 1. 字体浏览器
- 浏览系统中所有已安装的字体
- 实时字体预览
- 支持字体名称过滤搜索

### 2. 字体预览
- 自定义预览文本
- 可调节字体大小（12-120pt）
- 实时渲染效果展示

### 3. 字体比较
- 多字体并排比较
- 同步预览文本
- 最多支持4列同时比较

### 4. 字体搜索
- 按字体名称快速搜索
- 实时过滤结果
- 支持模糊匹配

### 5. 字体分类
- 自动分类：衬线、无衬线、等宽、手写体
- 手动标记收藏字体
- 按类别快速筛选

### 6. 字体导出
- 导出字体列表为 CSV 格式
- 导出字体列表为 TXT 格式
- 包含字体详细信息

### 7. 字体信息
- 显示字体族名称
- 显示字体样式
- 显示字体版本
- 显示字体文件路径

### 8. 自定义文本
- 输入任意文本进行预览
- 支持中英文混合
- 实时预览效果

## 安装要求

- Windows 10/11
- Python 3.10+
- PyQt6 6.5.0+

## 安装步骤

1. 克隆或下载本项目
```bash
cd FontManager
```

2. 安装依赖
```bash
pip install PyQt6
```

3. 运行程序
```bash
python main.py
```

## 使用说明

### 基本操作

1. **浏览字体**：左侧面板显示所有字体列表，点击即可预览
2. **搜索字体**：在搜索框中输入字体名称进行过滤
3. **调节大小**：使用滑块调节预览字体大小
4. **自定义文本**：在文本框中输入自定义预览内容

### 字体比较

1. 点击"字体比较"标签页
2. 从下拉菜单中选择要比较的字体
3. 输入比较文本
4. 点击"添加到比较"按钮

### 字体导出

1. 点击"导出字体列表"按钮
2. 选择导出格式（CSV 或 TXT）
3. 选择保存位置
4. 完成导出

### 字体分类

1. 切换到"字体分类"标签页
2. 选择分类筛选条件
3. 查看分类后的字体列表

## 技术栈

- **GUI 框架**：PyQt6
- **字体处理**：QFontDatabase
- **文件导出**：CSV 模块
- **主题样式**：QSS 样式表
