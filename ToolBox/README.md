# 🧰 ToolBox — HermesMy 工具箱

一站式工具管理平台，集成所有 HermesMy 工具的统一启动器。

## 功能特性

- 🎨 精美深色主题界面
- 📋 卡片式工具展示，直观清晰
- 🚀 一键启动任意工具
- ℹ️ 内置关于页面与版本信息

## 包含工具

| 工具 | 图标 | 说明 |
|------|------|------|
| **CleanMaster** | 🧹 | 系统清理大师 — 清理垃圾文件、释放磁盘空间 |
| **PDFMaster** | 📄 | PDF 处理专家 — 合并、拆分、转换 PDF |
| **ImageBatch** | 🖼️ | 图片批处理 — 格式转换、尺寸调整、水印 |
| **FileOrganizer** | 📁 | 文件整理助手 — 智能分类、批量重命名 |

## 运行要求

- Python 3.10+
- PyQt6

## 快速启动

```bash
cd G:\hermesMy\ToolBox
pip install PyQt6
python main.py
```
