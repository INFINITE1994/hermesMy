# WebScraper - 网页抓取工具

一款基于 PyQt6 的现代化网页抓取桌面应用。

## 功能

- **URL 抓取器** - 获取并显示网页内容
- **链接提取器** - 提取页面所有链接
- **图片下载器** - 批量下载页面图片
- **文本提取器** - 从 HTML 提取纯文本
- **批量抓取** - 同时抓取多个 URL
- **HTML 查看器** - 带语法高亮的 HTML 查看
- **导出功能** - 支持 CSV/TXT/JSON 格式导出

## 安装

```bash
pip install PyQt6 requests beautifulsoup4
```

## 运行

```bash
python main.py
```

## 技术栈

- Python 3.10+
- PyQt6
- Requests
- BeautifulSoup4
