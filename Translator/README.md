# Translator - 专业桌面翻译工具

一款功能强大的桌面翻译应用程序，基于 PyQt6 构建，支持多语言翻译、词典查询、批量翻译等功能。

## ✨ 功能特性

### 🌐 核心翻译功能
- **多语言翻译** - 支持 28+ 种语言互译
- **自动检测** - 智能识别源语言
- **语言切换** - 一键交换源语言和目标语言
- **实时翻译** - 快速获取翻译结果

### 📖 词典功能
- **词性标注** - 显示单词的词性分类
- **同义词** - 提供相关词汇和同义词
- **例句展示** - 显示真实语境中的例句

### 📜 历史记录
- **自动保存** - 自动记录所有翻译
- **搜索功能** - 快速查找历史翻译
- **一键加载** - 双击快速恢复历史翻译

### ⭐ 收藏管理
- **收藏功能** - 保存常用翻译
- **搜索过滤** - 快速查找收藏内容
- **导出功能** - 支持导出为 JSON 或 TXT 格式

### 📦 批量翻译
- **多行翻译** - 支持批量翻译多行文本
- **逐行处理** - 每行独立翻译
- **结果汇总** - 一次性显示所有翻译结果

### 🛠️ 实用工具
- **一键复制** - 快速复制翻译结果
- **保存文件** - 将翻译保存到文件
- **字数统计** - 实时显示输入字符数
- **快捷键支持** - 提高操作效率

## 🎨 界面设计

- **深色主题** - 专业的深色界面设计
- **渐变配色** - 使用紫色渐变作为主色调 (#667eea → #764ba2)
- **卡片布局** - 现代化的卡片式界面
- **中文界面** - 全中文用户界面

## 🚀 快速开始

### 系统要求

- Windows 10/11
- Python 3.10 或更高版本

### 安装步骤

1. **克隆或下载项目**
   ```bash
   cd G:\hermesMy\Translator
   ```

2. **安装依赖**
   ```bash
   pip install PyQt6 requests
   ```

3. **运行程序**
   ```bash
   python main.py
   ```

### 使用 pip 安装（推荐）

```bash
cd G:\hermesMy\Translator
pip install -e .
```

程序会在用户目录下创建 `.translator` 文件夹用于存储数据：

└── favorites.json       # 收藏的翻译

## 💡 使用技巧

1. **快速翻译** - 输入文本后按 `Ctrl + Enter` 即可快速翻译
2. **语言切换** - 点击中间的 `⇄` 按钮可快速交换源语言和目标语言
3. **查看词典** - 翻译单词时会自动显示词典释义和例句
4. **批量处理** - 使用批量翻译标签页可以一次性翻译多行文本
5. **收藏管理** - 点击 ⭐ 按钮收藏常用翻译，支持导出功能

## 🐛 常见问题

### Q: 翻译失败怎么办？
A: 请检查网络连接，程序需要联网才能使用翻译功能。

### Q: 如何清空历史记录？
A: 在历史记录标签页中，点击 🗑️ 按钮即可清空所有历史记录。

### Q: 收藏的翻译在哪里？
A: 收藏的翻译保存在 `~/.translator/favorites.json` 文件中。

### Q: 支持离线翻译吗？
A: 目前不支持离线翻译，需要联网使用 Google Translate API。
