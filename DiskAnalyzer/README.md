# DiskAnalyzer - 磁盘空间分析器

一款功能强大的磁盘空间分析桌面应用程序，使用 PyQt6 构建，提供直观的可视化界面帮助您了解磁盘使用情况。

## ✨ 功能特性

| 功能 | 说明 |
|------|------|
| 🗺️ **Treemap 可视化** | 以矩形树图形式直观展示文件夹大小分布 |
| ☀️ **旭日图** | 圆形图表展示磁盘使用层级关系 |
| 📋 **文件列表** | 可排序的文件列表，支持按大小、时间排序 |
| 📁 **文件夹树** | 可展开的目录结构树形视图 |
| 🏆 **Top 100 大文件** | 快速查找占用空间最多的 100 个文件 |
| 📊 **文件类型分析** | 按类型分类统计（图片、视频、文档、代码等） |
| 🕐 **旧文件查找** | 查找指定天数内未访问的文件 |
| 📄 **导出报告** | 支持导出为 HTML 或 CSV 格式报告 |

## 🎨 界面设计

- **深色主题**: 背景 `#0a0a0a`，卡片 `#111122`
- **渐变强调色**: `#667eea` → `#764ba2`
- **中文界面**，简洁现代

## 📦 安装与运行

### 系统要求

- Python 3.10+
- Windows 10/11

### 安装依赖

```bash
pip install PyQt6 psutil
```

### 运行

```bash
cd DiskAnalyzer
python main.py
```

## 📁 项目结构

```
DiskAnalyzer/

```

## 🔧 使用说明

1. 启动程序后，在顶部选择或输入要扫描的磁盘/目录路径
2. 点击 **开始扫描** 按钮
3. 扫描完成后，切换不同标签页查看分析结果
4. 使用 **导出** 按钮保存 HTML 或 CSV 报告
