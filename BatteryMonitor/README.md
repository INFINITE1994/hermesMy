## ✨ 功能特性

### 📊 电池状态
- 实时显示电池电量百分比
- 充电/放电状态指示
- 剩余使用时间预估
- 电压、电流、功率实时监控
- 精美圆形进度指示器

### 💊 电池健康
- 电池健康度评估
- 设计容量 vs 当前容量对比
- 电池制造商信息
- 序列号、化学类型
- 磨损程度计算

### ⚡ 电源计划管理
- 快速切换电源方案（节能/平衡/高性能）
- 当前电源计划显示
- 一键切换功能

### 📈 充电历史
- 电量变化趋势图表
- 可选时间范围（5分钟/15分钟/30分钟/1小时）
- 统计数据（平均/最低/最高电量）
- 充电次数统计

### 🔔 电池告警
- 低电量告警（可自定义阈值）
- 充电完成提醒
- 多种告警动作：
  - 仅显示通知
  - 播放声音
  - 进入睡眠模式
  - 进入休眠模式
- 告警历史记录

### 🔌 功耗监控
- 实时功率显示
- 平均功率计算
- 峰值功率记录
- 累计能耗统计
- 高耗电进程列表

### 😴 睡眠定时器
- 多种操作模式：
  - 睡眠
  - 休眠
  - 关机
  - 锁定屏幕
- 自定义延迟时间
- 快捷预设按钮（5/15/30/60/120/180分钟）
- 实时倒计时显示

### 💾 数据导出
- 多种导出格式：
  - CSV 文件
  - TXT 文件
  - JSON 文件
- 自定义数据范围
- 选择性导出内容
- 数据预览功能

## 🚀 快速开始

### 环境要求

- Windows 10/11
- Python 3.10 或更高版本
- pip 包管理器

### 安装步骤

1. **克隆或下载项目**
   ```bash
   cd G:\hermesMy\BatteryMonitor
   ```

2. **创建虚拟环境（推荐）**
   ```bash
   python -m venv venv
   venv\Scripts\activate
   ```

3. **安装依赖**
   ```bash
   pip install -e .
   ```
   
   或者手动安装：
   ```bash
   pip install PyQt6 psutil
   ```

4. **运行程序**
   ```bash
   python main.py
   ```

## 🔧 使用说明

### 查看电池状态

1. 启动程序后，默认显示「电池状态」标签页
2. 圆形指示器显示当前电量
3. 右侧面板显示详细信息

### 设置电池告警

1. 切换到「电池告警」标签页
2. 勾选「启用低电量告警」
3. 设置告警阈值（默认 20%）
4. 选择告警动作

### 使用睡眠定时器

1. 切换到「睡眠定时」标签页
2. 选择操作类型（睡眠/休眠/关机/锁定）
3. 设置延迟时间
4. 点击「开始定时」
5. 或使用快捷按钮快速设置

### 导出数据

1. 切换到「数据导出」标签页
2. 选择导出格式（CSV/TXT/JSON）
3. 选择数据范围
4. 勾选要导出的内容
5. 点击「选择保存位置并导出」

## ⚠️ 注意事项

1. **管理员权限**
   - 部分功能（如电源计划切换）可能需要管理员权限
   - 如遇权限问题，请以管理员身份运行

2. **电池信息**
   - 某些电池信息可能因硬件差异而不可用
   - 虚拟机环境可能无法检测到电池

3. **功耗计算**
   - 功耗数据为估算值
   - 实际功耗可能因硬件配置而异

4. **睡眠定时器**
   - 休眠功能需要系统支持
   - 关机操作会立即执行，请谨慎使用

## 🛠️ 故障排除

### 程序无法启动

```bash
# 检查 Python 版本
python --version

# 检查依赖安装
pip list | grep PyQt6
pip list | grep psutil
```

### 电池信息显示异常

- 确保在真实 Windows 设备上运行（非虚拟机）
- 尝试以管理员身份运行

### 电源计划切换失败

- 确保以管理员身份运行
- 检查电源方案是否存在
