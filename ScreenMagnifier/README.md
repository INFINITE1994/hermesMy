# 🔍 屏幕放大镜 - ScreenMagnifier

一款功能强大的屏幕放大镜桌面工具，基于 PyQt6 开发，支持多种实用功能。

## ✨ 功能特性

| 功能 | 说明 | 快捷键 |
|------|------|--------|
| 🔍 **屏幕放大** | 实时放大屏幕内容，支持 1× ~ 20× 缩放 | `Ctrl+Alt+M` |
| 🎨 **取色器** | 从屏幕上拾取任意颜色，显示 HEX/RGB 值 | `Ctrl+Alt+C` |
| 📏 **标尺** | 屏幕像素标尺，可拖动定位，可调长度 | `Ctrl+Alt+R` |
| ✚ **十字线** | 精准对齐十字准线 | 界面按钮切换 |
| 🔲 **网格覆盖** | 全屏半透明网格，方便对齐 | `Ctrl+Alt+G` |
| 📸 **截图** | 捕获放大区域，保存或复制到剪贴板 | `Ctrl+Alt+S` |
| ⌨️ **全局热键** | 后台全局快捷键控制 | 见上表 |
| 📌 **系统托盘** | 最小化到系统托盘，后台运行 | `Ctrl+Alt+H` |

## 🎨 界面主题

- **深色主题**: 背景 `#0a0a0a`，卡片 `#111122`
- **渐变强调色**: `#667eea` → `#764ba2`
- **中文字体**: 微软雅黑

## 📦 安装

### 环境要求

- Python 3.10+
- Windows 10/11

### 安装步骤

```bash
# 克隆或下载项目
cd ScreenMagnifier

# 创建虚拟环境（推荐）
python -m venv .venv
.venv\Scripts\activate

# 安装依赖
pip install PyQt6 Pillow pynput

# 或使用 uv
uv pip install PyQt6 Pillow pynput
```

## 🚀 运行

```bash
python main.py
```

启动后：
- 主窗口显示放大镜视图
- 鼠标滚轮调整缩放倍率
- 关闭窗口会最小化到系统托盘
- 双击托盘图标恢复窗口

## 📖 使用说明

### 放大镜
- 启动后自动实时放大鼠标周围的屏幕内容
- 使用鼠标滚轮或界面按钮调整倍率 (1× ~ 20×)
- 可切换十字线和网格叠加显示

### 取色器
- 点击「拾取颜色」按钮进入取色模式
- 屏幕变暗，显示放大镜和颜色信息
- 左键点击确认取色，右键或 ESC 取消
- 点击「复制色值」将 HEX 色值复制到剪贴板

### 标尺
- 点击「显示标尺」打开屏幕标尺
- 标尺可自由拖动到任意位置
- 可调整标尺长度 (100px ~ 1920px)

### 网格覆盖
- 点击「显示网格」在全屏显示半透明网格
- 可调整网格间距 (10px ~ 200px)
- 网格不阻挡鼠标操作

### 截图
- 「保存截图」将当前放大区域保存为 PNG/JPG
- 「复制到剪贴板」将截图直接复制到系统剪贴板

## 📝 快捷键汇总

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Alt+M` | 打开/激活放大镜窗口 |
| `Ctrl+Alt+C` | 启动取色器 |
| `Ctrl+Alt+R` | 切换标尺显示 |
| `Ctrl+Alt+G` | 切换网格覆盖 |
| `Ctrl+Alt+S` | 截图保存 |
| `Ctrl+Alt+H` | 显示/隐藏主窗口 |
| `Ctrl+Q` | 退出程序 |
| `Escape` | 关闭当前窗口 |
| 鼠标滚轮 | 调整放大倍率 |

## ⚠️ 注意事项

- 全局热键需要 `pynput` 库支持
- 高 DPI 屏幕下可能会有缩放差异
- 某些应用的 DRM 保护内容可能无法截取
