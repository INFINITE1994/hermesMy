# 剪贴板历史管理工具 (ClipboardHistory)

一款功能强大的 Windows 桌面剪贴板历史管理工具，基于 PyQt6 构建。

## 功能特性

- 📋 **文本历史** — 自动记录所有复制的文本内容及时间戳
- 🖼️ **图片历史** — 记录复制的图片，支持缩略图预览
- 🔍 **全文搜索** — 在历史记录中进行全文检索
- 🏷️ **智能分类** — 根据内容类型自动分类（URL、邮箱、代码、文本等）
- 📌 **收藏置顶** — 将常用片段置顶收藏
- ⚡ **快速粘贴** — 单击即可将历史片段粘贴到当前窗口
- 🔄 **同步导入导出** — 以 JSON 格式导出/导入历史记录
- ⌨️ **全局热键** — `Ctrl+Alt+V` 显示/隐藏窗口
- 🎨 **暗色主题** — 精心设计的深色 UI

## 安装

```bash
pip install PyQt6 Pillow
```

## 运行

```bash
python main.py
```

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Alt+V` | 显示/隐藏主窗口 |
| `Ctrl+F` | 聚焦搜索框 |
| `Escape` | 隐藏窗口 |

## 系统托盘

程序最小化后会驻留在系统托盘，右键托盘图标可快速访问常用功能。

## 数据存储

历史记录保存在用户目录下的 `.clipboard_history` 文件夹中：

- `history.json` — 文本历史记录
- `images/` — 图片文件
