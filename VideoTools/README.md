# 🎬 VideoTools - 视频处理工具箱

一款功能强大的桌面视频处理工具，基于 PyQt6 开发，支持多种视频处理操作。

## ✨ 功能特性

### 1. 🔄 视频格式转换
- 支持 MP4、AVI、MKV、MOV、WebM 等主流格式互转
- 自动识别源格式，一键转换

### 2. ✂️ 视频裁剪
- 精确设置开始时间和结束时间
- 支持 HH:MM:SS 格式输入

### 3. 🔗 视频合并
- 支持同时选择多个视频文件
- 按添加顺序合并为一个视频

### 4. 📦 视频压缩
- 自定义压缩比例（1%-100%）
- 有效减小视频文件体积

### 5. 🎵 音频提取
- 从视频中提取音频轨道
- 支持导出为 MP3、WAV 格式

### 6. ℹ️ 视频信息查看
- 显示视频详细元数据
- 包括分辨率、时长、比特率、编码器等

### 7. 📋 批量转换
- 一次性选择多个视频文件
- 统一转换为目标格式

## 🛠️ 安装

### 前置要求
- Python 3.10 或更高版本
- FFmpeg（moviepy 依赖）

### 安装步骤

```bash
# 克隆或下载项目
cd VideoTools

# 安装依赖
pip install PyQt6 moviepy

# 或使用 pyproject.toml
pip install .
```

### FFmpeg 安装
moviepy 需要 FFmpeg 支持：

1. 访问 https://ffmpeg.org/download.html 下载
2. 解压后将 bin 目录添加到系统 PATH 环境变量
3. 或使用 conda：`conda install ffmpeg`

## 🚀 运行

```bash
python main.py
```

## 📝 使用说明

1. **转换视频**：点击"视频转换"标签页，选择源文件和目标格式，点击开始转换
2. **裁剪视频**：设置开始时间和结束时间，选择源文件后执行裁剪
3. **合并视频**：添加多个视频文件，点击合并按钮
4. **压缩视频**：选择视频文件和压缩比例，开始压缩
5. **提取音频**：选择视频和音频格式，提取音频
6. **查看信息**：选择视频文件，自动显示详细信息
7. **批量转换**：添加多个文件，统一转换格式

## ⚠️ 注意事项

- 处理大文件时请耐心等待，进度条会显示处理进度
- 确保有足够的磁盘空间存放输出文件
- FFmpeg 必须正确安装才能使用所有功能
