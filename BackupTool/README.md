# 🛡️ BackupTool - 专业数据备份工具

一款功能强大的桌面数据备份应用程序，基于 PyQt6 构建，提供完整的企业级备份解决方案。

## ✨ 核心功能

### 📁 文件备份
- 支持文件和文件夹的完整备份
- 拖拽添加备份源
- 批量选择备份目标

### 🔄 增量备份
- 智能检测文件变更（基于修改时间和文件大小）
- 仅备份新增和修改的文件，大幅节省时间和存储空间
- 支持基于哈希校验的精确比对

### ⏰ 定时备份
- 灵活的备份调度：每日、每周、每月
- 自定义备份时间
- 后台静默执行，不影响正常使用

### 📦 压缩备份
- 支持 ZIP 压缩格式
- 可调节压缩级别（1-9）
- 显示压缩比率和节省空间

### 🔐 加密备份
- AES-256 加密算法保护备份数据
- 密码保护备份文件
- 安全的密钥派生机制

### 📋 备份历史
- 完整的备份操作记录
- 按时间、状态、类型筛选
- 支持清理历史记录

### ♻️ 数据恢复
- 从任意备份点恢复数据
- 支持选择性恢复（选择特定文件）
- 恢复前预览备份内容

### ☁️ 云同步
- 支持 OneDrive 云存储同步
- 支持 Google Drive 云存储同步
- 自动检测本地云盘文件夹

## 🖥️ 系统要求

- **操作系统**: Windows 10/11
- **Python**: 3.10 或更高版本
- **依赖**: PyQt6, psutil

## 📦 安装与运行

### 1. 安装依赖
```bash
pip install PyQt6 psutil
```

### 2. 运行程序
```bash
python main.py
```

### 3. 使用 uv（推荐）
```bash
uv run main.py
```
