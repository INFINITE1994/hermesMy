# 🤖 AutomationTool - 专业桌面自动化工具

一款功能强大的 Windows 桌面自动化工具，基于 PyQt6 构建，提供多种自动化功能模块。

## ✨ 功能特性

### 1. 🎬 宏录制器
- 录制鼠标点击、移动和键盘操作
- 支持回放录制的宏
- 可保存和加载宏文件

### 2. ⏰ 任务调度器
- 设置定时执行的任务
- 支持一次性、每天、每周调度
- 任务列表管理与状态监控

### 3. 📁 文件监控器
- 实时监控文件夹变化
- 文件创建、修改、删除事件触发
- 可配置触发的自动化操作

### 4. ⌨️ 快捷键管理器
- 自定义全局快捷键
- 快捷键绑定自动化操作
- 支持组合键设置

### 5. 📝 文本扩展器
- 设置缩写词自动扩展为完整文本
- 支持变量插入（日期、时间等）
- 快速管理文本片段

### 6. 🖱️ 自动点击器
- 可配置点击间隔（毫秒精度）
- 支持左键、右键、中键
- 可设置点击次数或持续运行

### 7. 📋 批量重命名
- 支持正则表达式替换
- 序号填充重命名
- 大小写转换、前后缀添加
- 实时预览重命名结果

### 8. 🔧 工作流构建器
- 可视化拖拽式工作流设计
- 支持多种节点类型
- 工作流保存与执行

## 🛠️ 安装与运行

### 环境要求
- Python 3.10+
- Windows 10/11

### 安装依赖
```bash
pip install PyQt6 psutil
```

### 运行程序
```bash
python main.py
```

## 📂 项目结构

```
AutomationTool/
├── main.py              # 程序入口
├── pyproject.toml       # 项目配置
├── README.md            # 项目说明
└── app/
    ├── __init__.py
    ├── main_window.py   # 主窗口
    ├── styles.py        # 主题样式
    └── modules/
        ├── __init__.py
        ├── macro_recorder.py    # 宏录制器
        ├── task_scheduler.py    # 任务调度器
        ├── file_watcher.py      # 文件监控器
        ├── hotkey_manager.py    # 快捷键管理器
        ├── text_expander.py     # 文本扩展器
        ├── auto_clicker.py      # 自动点击器
        ├── batch_rename.py      # 批量重命名
        └── workflow_builder.py  # 工作流构建器
```
