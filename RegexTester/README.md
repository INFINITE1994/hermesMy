# ⚡ RegexTester — 正则表达式测试工具

一款专业的正则表达式测试桌面应用，基于 Python 3.10+ 和 PyQt6 构建。

## ✨ 功能特性

| 功能 | 说明 |
|------|------|
| 🔤 **正则编辑器** | 支持正则语法高亮显示，实时预览 |
| 📝 **测试字符串** | 带行号的文本编辑器，匹配项高亮标记 |
| 🎯 **匹配结果** | 表格展示所有匹配，含分组、位置、长度详情 |
| 🔄 **查找替换** | 正则表达式查找替换，支持反向引用和命名组 |
| 📚 **常用模式** | 内置 20+ 常用正则模式（邮箱、手机、URL、IP 等） |
| 📖 **正则参考** | 完整的正则语法速查表 |
| 🔍 **中文解释** | 逐段解析正则表达式，以中文说明每个部分的含义 |
| 💾 **保存模式** | 保存和管理自定义正则模式 |

## 🎨 界面设计

- 暗色主题，护眼舒适
- 渐变色强调按钮（#667eea → #764ba2）
- 卡片式布局，层次分明
- 全中文界面

## 📦 安装

### 环境要求

- Python 3.10 或更高版本
- Windows / macOS / Linux

### 安装依赖

```bash
pip install PyQt6
```

或使用项目配置：

```bash
cd RegexTester
pip install -e .
```

## 🔧 使用技巧

1. **快速测试**：在顶部输入正则表达式，在下方输入测试文本，匹配结果会实时显示
2. **标志切换**：使用 `i` `m` `s` `x` 复选框切换正则标志
3. **双击加载**：在「常用模式」或「保存模式」中双击即可加载
4. **解释功能**：切换到「解释」标签页，点击按钮查看正则的中文逐段解析
5. **数据持久化**：保存的模式存储在 `~/.regex_tester/patterns.json`
