# 🎵 AudioTools - 音频处理工具箱

一款功能强大的桌面音频处理工具箱，基于 PyQt6 构建，提供直观的图形界面。

## ✨ 功能特性

| 功能 | 描述 |
|------|------|
| 🎵 格式转换 | 在 MP3、WAV、FLAC、OGG、AAC 之间自由转换 |
| ✂️ 音频裁剪 | 按时间精确截取音频片段 |
| 🔗 音频合并 | 将多个音频文件合并为一个 |
| 🔊 音量调节 | 增大或减小音频音量（±30dB） |
| ℹ️ 音频信息 | 查看音频文件的详细元数据 |
| 📦 批量转换 | 一次性转换多个音频文件 |
| 🎙️ 音频录制 | 从麦克风录制音频 |

## 🎨 界面特色

- 深色主题设计，护眼舒适
- 渐变色强调元素（#667eea → #764ba2）
- 中文界面，操作直观
- 卡片式布局，层次分明

## 📋 系统要求

- **操作系统**: Windows 10/11
- **Python**: 3.10 或更高版本
- **FFmpeg**: 需要安装并添加到 PATH（用于 MP3/AAC/OGG 格式支持）

## 🚀 安装步骤

### 1. 克隆或下载项目

```bash
cd G:\hermesMy\AudioTools
```

### 2. 安装依赖

```bash
pip install PyQt6 pydub sounddevice numpy soundfile
```

### 3. 安装 FFmpeg

1. 访问 https://ffmpeg.org/download.html 下载 FFmpeg
2. 解压到目录（如 `C:\ffmpeg`）
3. 将 `C:\ffmpeg\bin` 添加到系统 PATH 环境变量

### 4. 运行程序

```bash
python main.py
```

## 💡 使用说明

### 格式转换
1. 点击「格式转换」标签页
2. 选择输入文件
3. 设置输出格式和参数
4. 点击「开始转换」

### 音频裁剪
1. 选择音频文件
2. 设置开始和结束时间（秒）
3. 点击「开始裁剪」

### 音频合并
1. 添加多个音频文件
2. 调整文件顺序
3. 选择输出路径
4. 点击「开始合并」

### 音量调节
1. 选择音频文件
2. 使用滑块或预设按钮调节音量
3. 点击「应用音量调节」

### 批量转换
1. 添加多个文件或整个文件夹
2. 选择输出格式和目录
3. 点击「开始批量转换」

### 音频录制
1. 选择输入设备
2. 设置采样率和声道数
3. 点击「开始录制」
4. 录制完成后点击「停止」
5. 选择保存路径和格式，点击「保存录音」

## ⚠️ 注意事项

1. **FFmpeg 必需**: 转换 MP3、AAC、OGG 格式需要 FFmpeg 支持
2. **WAV 格式**: WAV 格式不需要 FFmpeg，可直接使用
3. **录制权限**: 使用麦克风录制需要系统授权
4. **文件编码**: 建议使用英文路径，避免中文路径可能导致的问题

## 🐛 常见问题

**Q: 提示 "pydub 未安装"**
A: 运行 `pip install pydub`

**Q: 转换 MP3 失败**
A: 请确保已安装 FFmpeg 并添加到 PATH

**Q: 录制没有声音**
A: 检查麦克风权限和默认录音设备设置

**Q: 程序无法启动**
A: 确保 Python 版本 ≥ 3.10，且 PyQt6 已正确安装
