# PDFMaster - 专业PDF处理工具

> 全功能PDF处理桌面应用，深色主题，现代化界面

## 功能特性

| 功能 | 说明 |
|------|------|
| **合并PDF** | 将多个PDF文件合并为一个 |
| **拆分PDF** | 按页码范围拆分PDF文件 |
| **压缩PDF** | 减小PDF文件体积 |
| **添加水印** | 为PDF页面添加文字水印 |
| **转为图片** | 将PDF页面导出为PNG/JPG图片 |
| **提取文本** | 从PDF中提取全部文字内容 |

## 安装

```bash
pip install PyQt6 PyPDF2 Pillow pdf2image
```

## 运行

```bash
python main.py
```

## 系统要求

- Windows 10/11
- Python 3.10+

## 技术栈

- **PyQt6** - 跨平台GUI框架
- **PyPDF2** - PDF读写操作
- **Pillow** - 图像处理
- **pdf2image** - PDF转图片（需poppler）

## 截图

深色主题界面，支持拖拽添加文件，渐变色按钮，圆角卡片布局。
