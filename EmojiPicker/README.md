# 🎨 Emoji 选择器

一款精致的桌面 Emoji 选择器工具，基于 PyQt6 构建，支持搜索、分类、收藏、最近使用等功能。

## ✨ 功能特性

- **Emoji 网格** — 全量 Emoji 展示，网格布局，直观浏览
- **🔍 搜索** — 支持中文名称和英文关键词搜索
- **📂 分类浏览** — 10 大分类：表情、手势、动物、食物、活动、旅行、物品、符号、旗帜
- **🕐 最近使用** — 自动记录最近使用的 Emoji（最多 50 个）
- **❤️ 收藏夹** — 右键添加/取消收藏，持久化存储
- **📋 一键复制** — 单击复制到剪贴板，支持系统通知
- **🖐️ 肤色选择** — 支持 6 种肤色修饰符
- **ℹ️ Emoji 信息** — 鼠标悬停显示名称、分类、Unicode 编码
- **🔔 系统托盘** — 关闭窗口后最小化到托盘，双击重新打开

## 🎨 界面主题

- 深色背景：`#0a0a0a`
- 卡片背景：`#111122`
- 强调渐变：`#667eea` → `#764ba2`
- 界面语言：中文

## 📦 安装

### 环境要求

- Python 3.10+
- Windows 10/11

### 安装步骤

```bash
# 克隆或下载项目
cd EmojiPicker

# 创建虚拟环境（推荐）
python -m venv .venv
.venv\Scripts\activate

# 安装依赖
pip install PyQt6 pyperclip
```

## 🚀 运行

```bash
python main.py
```

## 📖 使用说明

| 操作 | 说明 |
|------|------|
| 单击 Emoji | 复制到剪贴板 |
| 双击 Emoji | 复制到剪贴板 |
| 右键 Emoji | 添加/取消收藏 |
| 搜索框 | 输入关键词搜索 |
| 分类标签 | 切换分类 |
| 肤色下拉 | 选择肤色修饰符 |
| 关闭窗口 | 最小化到系统托盘 |
| 双击托盘图标 | 重新显示窗口 |
| 右键托盘图标 | 显示/退出 |

## 📁 数据存储

收藏和最近使用的数据保存在：

```
%APPDATA%\EmojiPicker\
├── favorites.json    # 收藏列表
└── recent.json       # 最近使用记录
```
