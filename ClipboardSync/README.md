# 📋 ClipboardSync - 剪贴板同步工具

一款功能强大的跨设备剪贴板同步工具，支持历史记录、搜索、分类、加密等功能。

## ✨ 主要功能

### 🔗 跨设备同步
- 支持多设备间剪贴板内容同步
- 设备标识管理
- 实时同步状态显示

### 📜 历史记录
- 自动保存剪贴板历史
- 时间戳显示（今天/昨天/具体日期）
- 最大支持10000条记录

### 🔍 智能搜索
- 全文搜索剪贴板内容
- 分类筛选
- 置顶内容快速访问

### 🏷️ 自动分类
- **文本** - 普通文本内容
- **链接** - URL网址自动识别
- **邮箱** - 邮箱地址识别
- **代码** - 代码片段识别
- **数字** - 纯数字内容
- **图片** - 图片内容
- **文件** - 文件路径

### 📌 置顶收藏
- 一键置顶常用内容
- 置顶内容独立筛选
- 快速复制访问

### 📋 快速粘贴
- 点击即复制到剪贴板
- 复制成功状态提示
- 剪贴板变化自动监听

### 🔐 数据加密
- AES加密保护敏感数据
- 密码保护
- 加密状态标识

### ⌨️ 全局热键
- `Ctrl+Shift+V` 快速显示/隐藏窗口
- 最小化到系统托盘
- 托盘图标快捷操作

## 🚀 快速开始

### 环境要求
- Python 3.10+
- Windows 10/11
- PyQt6
- Pillow

### 安装步骤

1. **克隆或下载项目**
   ```bash
   cd G:\hermesMy\ClipboardSync
   ```

2. **安装依赖**
   ```bash
   pip install PyQt6 Pillow
   ```

3. **运行程序**
   ```bash
   python main.py
   ```

### 使用pyproject.toml安装（推荐）

```bash
cd G:\hermesMy\ClipboardSync
pip install -e .
```

安装后可直接使用命令：
```bash
clipboardsync
```

## 📖 使用指南

### 基本操作

1. **启动程序** - 运行后自动监听剪贴板
2. **复制内容** - 任何复制操作都会被记录
3. **查看历史** - 主界面显示所有历史记录
4. **搜索内容** - 使用顶部搜索框
5. **筛选分类** - 左侧边栏选择分类
6. **置顶内容** - 点击📌图标置顶
7. **复制内容** - 点击📋图标复制到剪贴板
8. **删除记录** - 点击🗑图标删除

### 系统托盘

- **双击托盘图标** - 显示/隐藏主窗口
- **右键托盘图标** - 显示菜单
  - 显示主窗口
  - 退出程序

### 全局热键

- **Ctrl+Shift+V** - 全局显示/隐藏窗口

### 设置选项

点击左侧边栏"⚙️ 设置"按钮：

- **通用设置**
  - 开机自启动
  - 最大历史条数（100-10000）

- **加密设置**
  - 启用/禁用加密
  - 设置加密密码

- **同步设置**
  - 启用跨设备同步
  - 设备标识设置

### 导出数据

点击"📤 导出"按钮可将历史记录导出为JSON文件。

## 🐛 常见问题

### Q: 程序无法启动？
A: 确保已安装Python 3.10+和PyQt6：
```bash
pip install PyQt6
```

### Q: 剪贴板监听不工作？
A: 检查是否有其他程序占用剪贴板，尝试重启程序。

### Q: 如何设置开机自启？
A: 在设置中勾选"开机自启动"选项。

### Q: 加密密码忘记了？
A: 目前不支持密码恢复，请重新设置密码。历史加密数据将无法解密。

### Q: 如何彻底退出程序？
A: 右键系统托盘图标，选择"退出"。
