# 🔢 UnitConverter - 专业单位换算器

一款现代化、专业的桌面单位换算应用程序，基于 PyQt6 构建，采用深色主题设计。

![Python](https://img.shields.io/badge/Python-3.10+-blue?logo=python)
![PyQt6](https://img.shields.io/badge/PyQt6-6.4+-green?logo=qt)
![License](https://img.shields.io/badge/License-MIT-yellow)

## ✨ 功能特性

### 📏 支持 9 大类单位换算

| 类别 | 支持单位 |
|------|---------|
| 📏 **长度** | 千米、米、厘米、毫米、英里、码、英尺、英寸 |
| ⚖️ **重量** | 千克、克、毫克、吨、磅、盎司 |
| 🌡️ **温度** | 摄氏度、华氏度、开尔文 |
| 📐 **面积** | 平方米、平方千米、公顷、英亩、平方英尺 |
| 🧪 **体积** | 升、毫升、加仑、杯、立方米 |
| 🚀 **速度** | 千米/时、米/秒、英里/时、节 |
| 💾 **数据** | 字节、KB、MB、GB、TB |
| ⏰ **时间** | 秒、分、时、天、周、月、年 |
| 💱 **汇率** | 美元、欧元、人民币、日元、英镑（实时汇率） |

### 🎨 界面特点

- **深色主题**：护眼的深色配色方案（背景 #0a0a0a）
- **渐变高亮**：紫色渐变主题色（#667eea → #764ba2）
- **分类侧边栏**：直观的分类导航
- **即时换算**：输入即转换，实时显示结果
- **一键交换**：快速交换单位
- **中文界面**：完全中文化的用户界面

## 📦 安装

### 方式一：使用 pip 安装依赖

```bash
# 确保 Python 版本 >= 3.10
python --version

# 安装依赖
pip install PyQt6 requests

# 运行应用
python main.py
```

### 方式二：使用项目配置安装

```bash
# 克隆或下载项目
cd UnitConverter

# 安装项目及依赖
pip install -e .

# 运行
unitconverter
# 或
python main.py
```

## 🚀 使用方法

1. 运行 `python main.py` 启动应用
2. 在左侧侧边栏选择要换算的类别（长度、重量、温度等）
3. 输入数值
4. 选择"从"和"到"的单位
5. 点击转换按钮或直接查看结果（自动转换）
6. 使用 ⇄ 按钮快速交换单位

### 💱 汇率换算说明

- 汇率数据通过 API 实时获取
- 首次使用时需要网络连接
- 如果网络不可用，将使用默认汇率
