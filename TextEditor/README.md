# TextEditor - 轻量级代码编辑器

一个基于 PyQt6 构建的现代化代码编辑器，支持语法高亮、多标签页、代码缩略图等功能。

## ✨ 功能特性

### 🎨 语法高亮
支持以下语言的语法高亮：
- **Python** - 关键字、函数、类、装饰器、字符串、注释
- **JavaScript** - 关键字、内置对象、箭头函数、模板字符串
- **HTML** - 标签、属性、注释、实体字符
- **CSS** - 选择器、属性、值、媒体查询
- **JSON** - 键、值、数字、布尔值
- **Markdown** - 标题、粗体、斜体、链接、代码块

### 📑 多标签页
- 同时编辑多个文件
- 标签页可拖拽排序
- 关闭前自动提示保存
- 显示文件修改状态（● 标记）

### 📍 行号显示
- 实时行号显示
- 当前行高亮
- 自动调整宽度

### 🔍 查找替换
- 支持普通文本搜索
- 支持正则表达式
- 区分大小写选项
- 全词匹配选项
- 全部替换功能

### 💾 自动保存
- 每 30 秒自动保存已修改的文件
- 退出时提示保存更改

### 🔤 编码支持
- UTF-8（默认）
- GBK
- ASCII
- 自动检测编码

### 📝 自动换行
- 可切换的自动换行模式
- 快捷键：`Alt + Z`

### 📁 最近文件
- 记录最近打开的 15 个文件
- 快速重新打开
- 可清除历史记录

### 📤 导出功能
- 导出为带语法高亮的 HTML 文件
- 保留原始代码颜色和格式
- 包含行号显示

### 🗺️ 代码缩略图
- 右侧显示代码缩略图
- 可视化当前视口位置
- 支持点击跳转
- 可拖拽滚动

### ⌨️ 快捷键
| 快捷键 | 功能 |
|--------|------|
| `Ctrl + N` | 新建文件 |
| `Ctrl + O` | 打开文件 |
| `Ctrl + S` | 保存 |
| `Ctrl + Shift + S` | 另存为 |
| `Ctrl + W` | 关闭标签页 |
| `Ctrl + F` | 查找替换 |
| `Ctrl + Z` | 撤销 |
| `Ctrl + Y` | 重做 |
| `Ctrl + Shift + E` | 导出 HTML |
| `Alt + Z` | 切换自动换行 |
| `Tab` | 增加缩进 |
| `Shift + Tab` | 减少缩进 |

## 🎨 主题设计

采用深色主题，护眼且专业：
- **背景色**: `#0a0a0a` - 深黑色
- **卡片背景**: `#111122` - 深蓝黑色
- **强调色**: `#667eea` → `#764ba2` - 蓝紫渐变
- **代码颜色**: Material Design 风格语法高亮

## 🚀 安装与运行

### 系统要求
- Python 3.10 或更高版本
- Windows 10/11
- 推荐 4GB+ 内存

### 安装依赖
```bash
pip install PyQt6
```

### 运行程序
```bash
# 直接运行
python main.py

# 打开指定文件
python main.py path/to/file.py
```

### 使用 uv 安装（推荐）
```bash
cd TextEditor
uv pip install -e .
texteditor
```

```
TextEditor/
```

## 🔧 配置文件

程序会在用户目录下创建配置文件：
```
~/.texteditor_config.json
```

配置内容包括：
- 最近打开的文件列表
- 用户偏好设置
