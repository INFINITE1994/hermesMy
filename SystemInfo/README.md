# 系统信息查看器 (SystemInfo)

一个基于 PyQt6 的专业系统信息查看工具，用于查看 Windows 系统的详细硬件和软件信息。

## 功能特性

- **操作系统信息** - Windows 版本、构建号、架构
- **CPU 信息** - CPU 型号、核心数、线程数、频率、缓存
- **内存信息** - 内存大小、类型、速度、插槽使用情况
- **磁盘信息** - 磁盘型号、容量、类型、健康状态
- **GPU 信息** - 显卡型号、显存、驱动版本
- **网络信息** - 网络适配器、IP 地址、MAC 地址
- **主板信息** - 制造商、型号、BIOS 版本
- **导出报告** - 支持导出为 HTML 或 TXT 格式

## 安装要求

- Python 3.10+
- Windows 操作系统

## 安装依赖

```bash
pip install PyQt6 psutil GPUtil
```

或使用项目配置：

```bash
pip install .
```

## 运行程序

```bash
python main.py
```

## 界面特性

- 深色主题设计
- 渐变色强调效果 (#667eea → #764ba2)
- 中文界面
- 卡片式布局
- 响应式设计

## 技术栈

- **PyQt6** - 跨平台 GUI 框架
- **psutil** - 系统信息获取
- **GPUtil** - GPU 信息获取

## 导出功能

支持将系统信息导出为：
- HTML 格式报告（带样式）
- TXT 格式纯文本
