# SystemCleaner

**Windows 系统优化清理工具** — 一站式系统维护与优化解决方案

## 功能特性

| 模块 | 说明 |
|------|------|
| 🚀 启动项管理 | 管理开机自启动程序，禁用/启用启动项 |
| ⚙️ 服务管理 | 查看和管理 Windows 系统服务 |
| 📋 注册表清理 | 扫描并清理无效的注册表项 |
| 🔧 驱动更新 | 检查硬件驱动程序更新状态 |
| 💾 磁盘碎片整理 | 分析和整理磁盘碎片 |
| 🧠 内存优化 | 实时监控内存使用，一键释放内存 |
| 🌐 网络优化 | 优化网络参数设置，刷新 DNS 缓存 |
| 🔒 隐私清理 | 清理浏览器缓存、临时文件等隐私痕迹 |

## 安装

```bash
pip install PyQt6 psutil
```

## 运行

```bash
python main.py
```

## 技术栈

- Python 3.10+
- PyQt6 — 现代化 GUI 框架
- psutil — 系统监控库
- winreg — Windows 注册表操作

## 系统要求

- Windows 10 / 11
- Python 3.10 或更高版本
- 管理员权限（部分功能需要）
