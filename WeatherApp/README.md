# 🌤️ 天气预报 (WeatherApp)

一款精美的桌面天气预报应用，基于 PyQt6 构建，使用 wttr.in 免费 API 获取天气数据。

![Python](https://img.shields.io/badge/Python-3.10+-blue?logo=python)
![PyQt6](https://img.shields.io/badge/PyQt6-6.5+-green?logo=qt)
![License](https://img.shields.io/badge/License-MIT-yellow)

## ✨ 功能特性

| 功能 | 说明 |
|------|------|
| 🌡️ **当前天气** | 温度、湿度、风速、气压、能见度等实时数据 |
| 📅 **7天预报** | 未来一周天气预报，包含最高/最低温度 |
| ⏰ **逐小时预报** | 未来24小时逐小时天气变化 |
| 🗺️ **天气地图** | 精美的天气可视化动画效果 |
| 🏙️ **多城市管理** | 添加、删除和切换多个城市 |
| ⚠️ **天气预警** | 根据天气条件自动生成预警信息 |
| 💨 **空气质量** | AQI指数和污染物详情 |
| 🌅 **日出日落** | 日出日落时间、月相显示 |

## 🎨 界面特色

- **深色主题**：护眼的深色配色方案
- **渐变色设计**：紫蓝渐变 (#667eea → #764ba2) 视觉效果
- **中文界面**：全中文用户界面
- **动画效果**：雨、雪、云、阳光天气粒子动画
- **圆形仪表盘**：空气质量等数据的可视化展示
- **日出弧线**：直观显示太阳位置和日照时长

## 📦 安装要求

- Python 3.10 或更高版本
- PyQt6 >= 6.5.0
- requests >= 2.31.0

## 🚀 快速开始

### 1. 安装依赖

```bash
# 使用 pip 安装
pip install PyQt6 requests

# 或使用 uv 安装
uv pip install PyQt6 requests
```

### 2. 运行应用

```bash
python main.py
```

### 3. 使用 pyproject.toml 安装

```bash
pip install .
```

安装后可直接使用命令：

```bash
weather-app
```

## 🔧 配置说明

应用配置文件保存在用户目录下：

```
~/.weather_app/
├── config.json      # 用户配置（城市列表等）
└── cache/           # 天气数据缓存
    └── *.json
```

- **config.json**：保存用户添加的城市列表和最后使用的城市
- **cache/**：缓存天气数据，网络断开时可使用缓存数据

## 🌐 API 说明

本应用使用 [wttr.in](https://wttr.in) 免费天气 API：

- **无需注册**：不需要 API Key
- **全球覆盖**：支持全球大部分城市
- **JSON 格式**：返回结构化的天气数据
- **中文支持**：支持中文城市名称查询

API 请求示例：
```
https://wttr.in/Beijing?format=j1
```

## 🎯 使用指南

1. **添加城市**：在左侧搜索框输入城市名称，点击"添加"按钮
2. **切换城市**：在左侧城市列表中点击城市名称
3. **查看天气**：右侧标签页显示不同天气信息
4. **刷新数据**：点击"刷新天气数据"按钮获取最新数据
5. **删除城市**：选中城市后点击"删除选中城市"按钮

## ⚠️ 注意事项

1. **网络连接**：需要网络连接才能获取天气数据
2. **API 限制**：wttr.in 是免费服务，请避免过于频繁的请求
3. **缓存机制**：网络断开时会自动使用缓存数据
4. **城市名称**：支持中文和英文城市名称
5. **空气质量**：基于能见度和湿度估算，仅供参考
