# 🔐 CryptoTools - 加密工具箱

一个功能齐全、界面精美的加密工具桌面应用程序，基于 PyQt6 开发。

## ✨ 功能特性

### 1. 🔒 文本加密
- **AES-256-CBC** - 高级加密标准，256位密钥
- **DES-CBC** - 数据加密标准（TripleDES）
- **RSA-2048** - 非对称加密算法
- 支持自定义密钥和初始向量 (IV)

### 2. 📁 文件加密
- 支持任意文件的加密和解密
- 使用 AES-256-CBC 算法
- 自动处理文件填充

### 3. 🔢 哈希计算器
- **MD5** - 消息摘要算法
- **SHA1** - 安全散列算法1
- **SHA256** - 安全散列算法256
- **SHA512** - 安全散列算法512
- **BLAKE2b** - BLAKE2哈希函数（大版本）
- **BLAKE2s** - BLAKE2哈希函数（小版本）
- 支持同时计算多种哈希值

### 4. 🔑 密码生成器
- 可自定义密码长度（4-128位）
- 可选字符类型：大写字母、小写字母、数字、特殊字符
- 支持批量生成（最多20个）
- 使用 `secrets` 模块确保密码安全性

### 5. 📝 Base64 编码/解码
- 支持文本的 Base64 编码
- 支持 Base64 字符串的解码
- 自动处理 UTF-8 编码

### 6. 🔍 JWT 解码器
- 解码 JWT Token 的 Header（头部）
- 解码 JWT Token 的 Payload（载荷）
- 自动处理 Base64URL 编码

### 7. 📜 证书查看器
- 支持 PEM 和 DER 格式的证书
- 显示证书详细信息：
  - 主题（Subject）
  - 颁发者（Issuer）
  - 序列号
  - 生效/过期日期
  - 签名算法
  - DNS 名称（SAN）

### 8. 🆔 UUID 生成器
- **UUID v1** - 基于时间戳和MAC地址
- **UUID v4** - 随机生成
- 支持自定义格式：
  - 大写/小写
  - 有/无连字符
  - 有/无花括号
- 支持批量生成（最多50个）

## 🎨 界面设计

- **深色主题**：护眼的深色配色方案
- **渐变强调色**：从 #667eea 到 #764ba2

## 📋 使用说明

### 文本加密
1. 选择加密算法（AES/DES/RSA）
2. 输入密钥（AES需要32字节，DES需要8字节）
3. 可选：输入初始向量（IV）
4. 输入要加密的文本
5. 点击"加密"按钮
6. 复制加密结果

### 文件加密
1. 点击"浏览"选择文件
2. 输入加密密钥
3. 点击"加密文件"
4. 加密后的文件将保存为 `.encrypted` 后缀

### 哈希计算
1. 输入要计算哈希的文本
2. 选择要使用的哈希算法
3. 点击"计算哈希值"
4. 查看并复制结果

### 密码生成
1. 设置密码长度
2. 选择字符类型
3. 设置生成数量
4. 点击"生成密码"
5. 复制生成的密码

## ⚠️ 安全提示

- 请妥善保管加密密钥，密钥丢失将无法解密
- RSA 加密时会生成新的密钥对，请保存好私钥
- 本工具仅供学习和测试使用，生产环境请使用专业的加密方案
