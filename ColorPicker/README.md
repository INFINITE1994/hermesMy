# 🎨 ColorPicker - 专业色彩拾取与调色板工具

一款功能强大的 PyQt6 桌面色彩工具，为设计师和开发者打造。

## ✨ 功能特性

### 🖱️ 屏幕取色器
- 从屏幕任意位置拾取颜色
- 放大镜功能，精确取色
- 实时预览拾取的颜色

### 🎡 色轮
- 交互式色轮选择
- HSV 滑块精确调节
- 实时颜色预览

### 📐 色彩格式
- HEX、RGB、HSL、HSV 格式显示
- 一键复制色彩值
- 格式实时转换

### 🎨 调色板生成器
- **互补色** - 180° 色相差
- **类似色** - 相邻色相
- **三角色** - 120° 等间距
- **分裂互补** - 互补色两侧
- **四角色** - 90° 等间距

### 📜 色彩历史
- 自动记录拾取的颜色
- 点击历史记录快速应用
- 最多保存 50 种颜色

### 📤 导出功能
- **CSS** 变量格式
- **SCSS** 变量格式
- **JSON** 数据格式
- **ASE** Adobe 色板格式

### ♿ 对比度检查器
- WCAG 2.1 标准检测
- AA / AAA 等级判定
- 实时对比度计算
- 文字预览效果

### 🌈 渐变生成器
- 线性渐变 / 径向渐变
- 多色标支持
- 角度调节
- CSS 代码一键复制

## 🛠️ 安装与运行

### 环境要求
- Python 3.10+
- Windows 10/11

### 安装依赖
```bash
pip install PyQt6
```

### 运行应用
```bash
python main.py
```

### 使用 uv 安装
```bash
uv pip install PyQt6
python main.py
```

## 🎯 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+P` | 启动屏幕取色 |
| `Ctrl+C` | 复制当前颜色值 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+G` | 生成渐变 |
| `Esc` | 取消取色 |
