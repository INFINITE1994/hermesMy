# TextTools 文本处理工具箱

一个功能丰富的文本处理工具箱桌面应用，基于 PyQt6 构建，采用现代化深色主题设计。

## ✨ 功能特性

| 工具 | 说明 |
|------|------|
| 📄 **文本格式转换** | 在 Markdown、HTML、纯文本之间互相转换 |
| 📋 **JSON 格式化** | JSON 格式化、验证、压缩 |
| 🔐 **Base64 编解码** | Base64 编码和解码 |
| 🌐 **URL 编解码** | URL 编码和解码 |
| #️⃣ **哈希生成器** | 生成 MD5、SHA1、SHA256 哈希值 |
| 🔍 **正则测试器** | 正则表达式测试，支持匹配高亮 |
| 📊 **文本对比** | 比较两段文本并显示差异 |
| 📝 **字数统计** | 统计字数、字符数、行数、段落数 |

## 🎨 界面设计

- **深色主题**：护眼的深色配色方案
- **渐变强调色**：#667eea → #764ba2
- **卡片式布局**：每个工具独立成卡片
- **中文界面**：完全中文化的用户界面

## 🚀 快速开始

### 环境要求

- Python 3.10 或更高版本
- Windows 10/11

### 安装依赖

```bash
pip install PyQt6
```

### 运行应用

```bash
python main.py
```

或者使用项目脚本：

```bash
pip install -e .
texttools
```

## 📝 使用说明

### 文本格式转换
在左侧面板输入文本，选择目标格式，点击转换按钮即可。

### JSON 格式化
粘贴 JSON 文本，可选择格式化（带缩进）、压缩（单行）或验证。

### Base64 编解码
输入文本后点击编码或解码按钮，支持中文字符。

### URL 编解码
处理 URL 中的特殊字符编码和解码。

### 哈希生成器
输入文本，一键生成 MD5、SHA1、SHA256 三种哈希值。

### 正则测试器
输入正则表达式和测试文本，实时显示匹配结果和高亮。

### 文本对比
在左右两个面板输入文本，点击对比查看差异，支持逐行对比。

### 字数统计
实时统计文本的字数、字符数（含/不含空格）、行数、段落数和句子数。
