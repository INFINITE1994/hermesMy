# 🧹 DiskCleanup - 专业磁盘清理工具

一款功能强大的 Windows 磁盘清理桌面应用程序，基于 PyQt6 构建，提供全面的系统清理和优化功能。

## ✨ 功能特性

### 1. 🗑️ 垃圾文件扫描
- 扫描系统临时文件
- 清理应用程序缓存
- 删除过期日志文件
- 智能识别可安全删除的文件

### 2. 🌐 浏览器清理
- 支持 Chrome、Edge、Firefox 等主流浏览器
- 清理浏览缓存、Cookie、历史记录
- 删除自动填充数据
- 清理下载记录

### 3. 🪟 Windows 系统清理
- 清理 Windows 更新缓存
- 删除系统临时文件
- 清理 Windows 日志
- 清理预取文件

### 4. 📦 应用程序清理
- 扫描已安装应用的缓存
- 清理应用临时数据
- 删除应用日志文件
- 支持自定义清理规则

### 5. 🔧 注册表清理
- 扫描无效注册表项
- 查找孤立的文件关联
- 检测损坏的 COM 组件引用
- 安全备份后再清理

### 6. 📁 重复文件查找
- 基于文件内容的精确匹配
- 使用 MD5 哈希算法
- 支持自定义扫描路径
- 智能推荐保留版本

### 7. 📊 大文件查找
- 快速扫描指定驱动器
- 按文件大小排序显示
- 支持文件类型过滤
- 一键定位大文件

### 8. ⏰ 定时清理
- 设置自动清理计划
- 支持每日、每周、每月任务
- 后台静默执行
- 清理日志记录

## 🎨 界面设计

- **深色主题**：护眼暗色界面

## 🚀 安装与运行

### 环境要求
- Windows 10/11
- Python 3.10+

### 安装步骤

```bash
# 克隆或下载项目
cd DiskCleanup

# 创建虚拟环境（推荐）
python -m venv .venv
.venv\Scripts\activate

# 安装依赖
pip install -e .

# 运行程序
python main.py
```

### 打包为可执行文件

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name DiskCleanup main.py
```

## ⚠️ 安全提示

- 注册表清理前会自动创建备份
- 建议在清理前检查扫描结果
- 定时清理任务可在设置中随时取消
- 系统关键文件会被自动跳过
