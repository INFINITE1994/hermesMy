# 高级进程管理器 (ProcessManager)

一款功能强大、界面精美的 Windows 进程管理工具，基于 PyQt6 构建。

## ✨ 功能特性

### 🔍 进程列表
- 实时显示所有运行中的进程
- 显示 CPU 和内存使用率
- 支持搜索和过滤进程
- 点击列头排序

### 🌳 进程树
- 以树形结构展示进程的父子关系
- 清晰了解进程层级结构

### ❌ 终止进程
- 支持强制终止选中的进程
- 安全的进程终止确认对话框

### 📋 进程详情
- 进程完整路径
- 命令行参数
- 打开的句柄数
- 线程数
- 创建时间等详细信息

### 📊 资源监控
- 实时 CPU 使用率曲线图
- 实时内存使用率曲线图
- 动态更新图表

### 🚀 启动项管理
- 查看系统启动程序
- 管理开机自启动项

### ⚙️ 服务管理
- 列出所有 Windows 服务
- 显示服务状态和启动类型
- 支持启动/停止/重启服务

### 📤 导出功能
- 导出进程列表为 CSV 格式
- 导出进程列表为 TXT 格式

## 🎨 界面设计

- 深色主题：背景色 #0a0a0a，卡片色 #111122
- 渐变强调色：#667eea 到 #764ba2
- 现代化 UI 设计
- 中文界面

```bash
pip install PyQt6 psutil
```

## 🔧 系统要求

- Windows 10/11
- Python 3.10+
- PyQt6 6.4.0+
- psutil 5.9.0+

## 📝 使用说明

1. **查看进程**：启动后自动加载所有进程，可通过搜索框过滤
2. **查看详情**：点击进程后在右侧面板查看详细信息
3. **终止进程**：选中进程后点击"终止进程"按钮
4. **切换视图**：使用左侧导航切换不同功能页面
5. **导出数据**：在进程列表页面点击导出按钮
