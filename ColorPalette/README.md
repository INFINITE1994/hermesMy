# 🎨 ColorPalette - 智能配色方案生成器

一款功能强大的桌面配色工具，帮助设计师和开发者快速创建、管理和导出配色方案。

## ✨ 功能特性

### 🤖 AI 配色
- 输入关键词（温暖、冷色、自然、海洋、日落等）自动生成配色方案
- 支持 12 种预设主题关键词
- 支持自定义关键词智能生成

### 🖼️ 图片提取
- 从上传的图片中自动提取主要颜色
- 使用 K-Means 聚类算法分析色彩分布
- 支持 JPG、PNG、BMP、WebP、GIF 格式
- 可自定义提取颜色数量（3-12个）

### 🎯 色彩和谐
- **互补色** - 180° 对比配色
- **类似色** - 相邻色彩和谐搭配
- **三角色** - 120° 等距三色方案
- **分裂互补** - 互补色的柔和变体
- **四角色** - 90° 等距四色方案
- **单色** - 同色系明暗变化

### 🔥 趋势配色
- Material Design 3
- Flat UI 经典配色
- Tailwind CSS 调色板
- Instagram 品牌渐变
- Spotify 品牌色彩
- GitHub 暗黑主题
- Glassmorphism 毛玻璃风格
- Cyberpunk 赛博朋克

### 💾 保存管理
- 收藏和组织自定义配色方案
- 支持标签分类（项目、品牌、网站、插画、UI设计）
- JSON 格式本地持久化存储
- 导入/导出配色方案

### 📤 多格式导出
- **CSS** - CSS 自定义属性格式
- **SCSS** - Sass 变量格式
- **JSON** - 结构化数据格式
- **ASE** - Adobe Swatch Exchange 格式

### 🔗 分享功能
- 生成 Coolors.co 在线链接
- 生成 Adobe Color 链接
- Base64 编码分享码

### 🔍 无障碍检查
- WCAG 2.1 对比度检测
- AA / AAA 等级判定
- 实时预览效果
- 大文字标准检测

## 🚀 安装与运行

### 环境要求
- Python 3.10+
- Windows 10/11

### 安装依赖
```bash
pip install PyQt6 Pillow
```

### 运行应用
```bash
python main.py
```

### 使用 uv 安装
```bash
uv pip install PyQt6 Pillow
python main.py
```

## 🎨 操作说明

1. **选择配色方式** - 通过顶部标签页选择 AI 配色、图片提取、色彩和谐或趋势配色
2. **生成配色** - 点击相应按钮或输入关键词生成配色方案
3. **编辑颜色** - 左键点击色块复制色值，右键点击可修改颜色
4. **保存收藏** - 点击「保存」按钮将当前配色方案添加到收藏
5. **导出使用** - 点击「导出」选择格式并保存或复制代码
6. **检查对比度** - 点击「对比度」检查颜色的无障碍等级

## 🛠️ 技术栈

- **PyQt6** - 跨平台 GUI 框架
- **Pillow** - 图像处理与颜色提取
- **colorsys** - 色彩空间转换
- **K-Means** - 聚类算法提取主色调
