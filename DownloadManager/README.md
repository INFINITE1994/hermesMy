# ⬇ 专业下载管理器 (DownloadManager)

一款基于 PyQt6 的专业多线程下载管理器，支持多线程分块下载、队列管理、速度限制等高级功能。

## ✨ 功能特性

### 🔀 多线程下载
- 将文件分割为多个块同时下载，大幅提升下载速度
- 支持 1-32 个并发线程，可根据网络环境自由调整
- 自动检测服务器是否支持断点续传（Range 请求）

### 📋 队列管理
- 添加、暂停、恢复、取消下载任务
- 支持最大并发下载数设置（1-20）
- 等待中的任务会在有空闲槽位时自动开始

### ⚡ 速度限制
- 可为每个下载任务单独设置速度限制
- 支持 KB/s 级别的精确限速
- 设置为 0 表示不限速

### 📊 实时进度追踪
- 实时显示下载进度百分比
- 显示当前下载速度
- 显示预计剩余时间（ETA）
- 显示已下载大小和总文件大小

### 📜 下载历史
- 自动保存下载完成和失败的记录
- 支持按文件名或 URL 搜索历史
- 最多保存 500 条历史记录

### 🔗 剪贴板监控
- 自动监控剪贴板中的下载链接
- 智能识别常见文件类型的下载链接
- 检测到链接后自动填入输入框

### 📦 批量下载
- 支持一次性输入多个 URL 进行批量下载
- 可从剪贴板批量粘贴链接
- 批量任务支持独立的线程数和限速设置

### 🔄 自动重试
- 下载失败后自动重试（默认最多 3 次）
- 可在设置中开启/关闭自动重试
- 手动重试不受次数限制

## 🚀 快速开始

### 环境要求

- Python 3.10 或更高版本
- Windows 10/11

### 安装依赖

```bash
pip install PyQt6 requests
```

### 运行程序

```bash
cd G:\hermesMy\DownloadManager
python main.py
```

### 使用 pip 安装（可选）

```bash
cd G:\hermesMy\DownloadManager
pip install -e .
```

安装后可直接使用命令：

```bash
download-manager
```

## 📖 使用指南

### 添加单个下载

1. 在顶部输入框粘贴或输入下载链接
2. 设置线程数（推荐 4-16）
3. 可选设置速度限制（0 表示不限速）
4. 选择保存目录
5. 点击「➕ 添加下载」按钮

### 批量下载

1. 切换到「📋 批量下载」标签页
2. 每行输入一个下载链接
3. 也可点击「📋 从剪贴板粘贴」按钮
4. 点击「➕ 批量添加」按钮

### 管理下载任务

- **暂停**：点击任务行的 ⏸ 按钮
- **继续**：点击已暂停任务的 ▶ 按钮
- **取消**：点击任务行的 ✖ 按钮
- **重试**：点击失败任务的 🔄 按钮
- **打开文件**：点击已完成任务的 📂 按钮
- **打开文件夹**：点击已完成任务的 📁 按钮
- **右键菜单**：提供更多操作选项

### 查看历史

1. 切换到「📜 历史记录」标签页
2. 使用搜索框按文件名或 URL 过滤
3. 可清空所有历史记录

## ⚙ 设置选项

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 最大并发下载数 | 同时进行的最大下载任务数 | 3 |
| 默认线程数 | 新建任务的默认线程数 | 8 |
| 默认保存目录 | 文件默认保存位置 | ~/Downloads |
| 监控剪贴板 | 自动检测剪贴板中的下载链接 | 开启 |
| 自动重试 | 失败后自动重试下载 | 开启 |
| User-Agent | HTTP 请求的 User-Agent | Chrome UA |
