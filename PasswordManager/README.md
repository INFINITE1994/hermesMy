# 🔐 安全密码管理器

一个基于 PyQt6 的本地密码管理器，使用 AES-256 加密保护您的所有密码。

## ✨ 功能特性

- **AES-256 加密** - 使用军事级别的加密算法保护您的密码
- **密码生成器** - 生成可自定义规则的强密码
- **分类管理** - 按类别组织密码（邮箱、社交、银行等）
- **快速搜索** - 快速搜索已保存的密码
- **一键复制** - 单击复制密码，30秒后自动清除剪贴板
- **导入/导出** - 支持 CSV 格式导入导出
- **密码强度指示器** - 可视化显示密码强度
- **自动锁定** - 不活动时自动锁定，保护数据安全

## 🚀 安装与运行

### 1. 安装依赖
```bash
pip install PyQt6 cryptography
```

### 2. 运行程序
```bash
python main.py
```

## 🛡️ 安全特性

- 使用 AES-256-CBC 加密算法
- PBKDF2 密钥派生（100,000 次迭代）
- 主密码不存储在本地
- 剪贴板内容 30 秒后自动清除
- 不活动时自动锁定
- 所有密码本地加密存储

## 📝 使用说明

1. 首次运行时设置主密码
2. 使用主密码登录
3. 添加、编辑或删除密码条目
4. 使用密码生成器创建强密码
5. 使用搜索功能快速查找密码
6. 通过分类功能组织密码

## ⚠️ 注意事项

- 请牢记您的主密码，忘记后无法恢复
- 建议定期备份密码数据
- 使用导出功能创建加密备份
