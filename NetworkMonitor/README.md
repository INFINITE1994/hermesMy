# NetworkMonitor - 网络监控工具

一个功能强大的实时网络监控桌面应用程序，基于 PyQt6 构建，提供专业的网络性能监控和分析功能。

## ✨ 功能特性

### 🚀 实时速度监控
- 实时显示上传/下载速度
- 精美仪表盘可视化展示
- 自动适应速度范围

### 📊 流量监控
- 监控总数据传输量
- 显示接收和发送的总字节数
- 实时更新统计数据

### 🔗 连接列表
- 列出所有活跃网络连接
- 显示进程名称、PID、协议类型
- 支持本地/远程地址查看

### 📈 带宽使用监控
- 按应用程序监控带宽使用
- 显示每个进程的接收/发送数据量
- 自动排序显示前20个进程

### 📉 速度历史图表
- 实时速度曲线图
- 支持下载/上传双曲线显示
- 最近2小时历史数据记录

### 🔔 告警系统
- 下载速度告警
- 上传速度告警
- 每日流量告警
- 系统托盘通知提醒

### 📥 数据导出
- 支持 CSV 格式导出
- 支持 TXT 格式导出
- 包含完整历史数据

### 🖥️ 系统托盘
- 最小化到系统托盘
- 右键菜单快速操作
- 双击恢复窗口

## 🎨 界面设计

- **深色主题**: 专业级暗色界面设计
- **渐变配色**: 使用 #667eea 到 #764ba2 的渐变色
- **卡片布局**: 现代化卡片式界面
- **响应式设计**: 自适应窗口大小

## 📋 系统要求

- **操作系统**: Windows 10/11
- **Python**: 3.10 或更高版本
- **依赖库**: 
  - PyQt6 >= 6.5.0
  - psutil >= 5.9.0

## 🚀 安装指南

### 1. 克隆或下载项目
```bash
cd G:\hermesMy\NetworkMonitor
```

### 2. 安装依赖
```bash
pip install PyQt6 psutil
```

或者使用项目配置安装：
```bash
pip install -e .
```

### 3. 运行程序
```bash
python main.py
```

## 📖 使用说明

### 启动应用
运行 `python main.py` 启动网络监控工具。

### 查看实时速度
- 主界面顶部显示实时上传/下载速度
- 仪表盘直观展示速度范围
- 速度卡片显示具体数值

### 查看连接列表
- 点击"连接列表"标签页
- 查看所有活跃网络连接
- 可按进程、协议等排序

### 查看带宽使用
- 点击"带宽使用"标签页
- 查看各应用程序的网络使用情况
- 按使用量自动排序

### 设置告警
- 点击"告警设置"按钮
- 配置速度和流量阈值
- 启用/禁用各类告警

### 导出数据
- 点击"导出数据"按钮
- 选择导出格式（CSV/TXT）
- 保存到指定位置

### 最小化到托盘
- 点击"最小化"按钮或关闭窗口
- 程序将最小化到系统托盘
- 双击托盘图标恢复窗口

## 🔧 配置说明

### 告警配置
- **下载速度告警**: 当下载速度超过设定阈值时触发
- **上传速度告警**: 当上传速度超过设定阈值时触发
- **流量告警**: 当每日流量超过设定阈值时触发

### 数据存储
- 速度历史数据保存在内存中
- 最多保存2小时的历史数据
- 导出时可保存完整历史记录

## 🐛 故障排除

### 程序无法启动
1. 检查 Python 版本是否 >= 3.10
2. 确认已安装 PyQt6 和 psutil
3. 检查是否有权限访问网络信息

### 无法获取连接信息
- 某些功能需要管理员权限
- 尝试以管理员身份运行程序

### 速度显示为0
- 确认网络连接正常
- 检查是否有网络活动
- 等待几秒钟让数据更新
