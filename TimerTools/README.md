# ⏱ TimerTools - 计时工具箱

一款功能丰富、界面精美的计时器与闹钟工具箱桌面应用程序。

使用 **PyQt6** 构建，采用深色主题设计，具有现代化的渐变配色方案。

---

## ✨ 功能特性

### 1. ⏱ 倒计时器
- 设置任意时长的倒计时
- 支持小时、分钟、秒数精确设置
- 快速预设按钮（1/5/10/15/25/30/60分钟）
- 实时进度条显示
- 倒计时结束时播放系统提示音

### 2. ⏱ 秒表
- 高精度计时（10毫秒分辨率）
- 开始/暂停/继续/重置操作
- 计圈功能，记录每圈用时
- 自动计算最快圈、最慢圈、平均圈
- 圈数差异对比显示

### 3. 🌍 世界时钟
- 显示本地时间
- 同时查看多个时区的当前时间
- 支持 20+ 个常用时区
- 可自由添加/删除时区
- 实时自动更新

### 4. 🍅 番茄钟
- 经典 25/5 分钟工作/休息循环
- 可自定义工作和休息时长
- 实时进度条显示
- 自动切换工作/休息模式
- 统计完成周期数和总专注时间
- 不同模式使用不同颜色提示

### 5. ⏰ 闹钟
- 设置多个闹钟
- 支持自定义闹钟标签
- 实时显示当前时间
- 闹钟触发时播放提示音
- 每分钟自动检查闹钟状态

### 6. 📅 日期计算器
- 计算两个日期之间的天数
- 日期加减计算（任意天数）
- 查询某天是星期几
- 显示相对日期（今天/明天/昨天）
- 支持弹出日历选择器

### 7. 🤝 会议规划
- 选择多个参与时区
- 自定义工作时间范围
- 分析 UTC 24 小时的可用性
- 高亮显示所有地区都可用的时间段
- 推荐最佳会议时间
- 无完美时段时给出最接近的建议

## 🚀 安装与运行

### 环境要求

- Python 3.10 或更高版本
- Windows 操作系统（用于播放提示音）

### 安装依赖

```bash
pip install PyQt6
```

### 运行应用

```bash
python main.py
```

## 🛠 技术栈

- **Python 3.10+**
- **PyQt6** - 跨平台 GUI 框架
- **winsound** - Windows 系统声音播放
- **datetime** - 日期时间处理
- **QTimer** - 定时器实现

---

## 📝 使用说明

1. 启动应用后，通过顶部选项卡切换不同工具
2. 每个工具都有独立的控制按钮和显示区域
3. 所有计时器支持暂停和继续
4. 闹钟和倒计时结束后会播放系统提示音
5. 世界时钟和番茄钟会自动更新显示
