# NetTools - 网络工具箱

一个功能齐全的网络工具箱桌面应用程序，使用 PyQt6 构建，提供多种常用网络诊断工具。

## 功能特性

### 1. Ping 工具
- Ping 任意主机并显示详细统计信息
- 支持自定义数据包数量和超时时间
- 实时显示延迟统计（最小/最大/平均/丢包率）

### 2. 端口扫描器
- 扫描指定主机的开放端口
- 支持自定义端口范围
- 多线程并发扫描，速度快
- 显示端口对应的服务名称

### 3. DNS 查询
- 解析域名到 IP 地址
- 支持多种 DNS 记录类型（A、AAAA、MX、CNAME、NS、TXT）
- 显示完整 DNS 记录信息

### 4. IP 信息
- 显示本地 IP 地址和网络接口信息
- 查询公网 IP 地址
- 显示地理位置和 ISP 信息

### 5. 路由追踪
- 追踪到目标主机的网络路径
- 显示每一跳的延迟和地理位置
- 支持自定义最大跳数

### 6. 网速测试
- 测试下载和上传速度
- 实时显示测试进度
- 显示 ping 延迟和抖动

### 7. Whois 查询
- 查询域名注册信息
- 显示注册商、注册日期、到期日期等
- 支持多种域名后缀

### 8. 网络扫描器
- 发现本地网络中的设备
- 显示设备 IP、MAC 地址和主机名
- 支持自定义扫描范围

## 安装要求

- Python 3.10 或更高版本
- Windows 10/11

## 安装步骤

1. 克隆或下载项目
2. 进入项目目录
3. 安装依赖：

```bash
pip install -e .
```

或直接安装依赖：

```bash
pip install PyQt6 psutil requests
```

## 运行应用

```bash
python main.py
```

## 使用说明

### Ping 工具
1. 输入目标主机地址
2. 设置数据包数量（默认 4 个）
3. 点击 "开始 Ping" 按钮

### 端口扫描器
1. 输入目标主机地址
2. 设置端口范围（默认 1-1024）
3. 点击 "开始扫描" 按钮

### DNS 查询
1. 输入要查询的域名
2. 选择记录类型
3. 点击 "查询" 按钮

### IP 信息
1. 点击 "获取本地 IP" 查看本地网络信息
2. 点击 "获取公网 IP" 查看公网信息

### 路由追踪
1. 输入目标主机地址
2. 设置最大跳数
3. 点击 "开始追踪" 按钮

### 网速测试
1. 点击 "开始测试" 按钮
2. 等待测试完成
3. 查看下载/上传速度结果

### Whois 查询
1. 输入要查询的域名
2. 点击 "查询" 按钮
3. 查看域名注册信息

### 网络扫描器
1. 输入网络地址范围（如 192.168.1.0/24）
2. 点击 "开始扫描" 按钮
3. 等待扫描完成，查看发现的设备

## 技术栈

- **GUI 框架**: PyQt6
- **网络工具**: socket, subprocess
- **系统信息**: psutil
- **HTTP 请求**: requests
- **并发处理**: threading, QThread
