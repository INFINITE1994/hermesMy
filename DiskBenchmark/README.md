# DiskBenchmark - 磁盘性能基准测试工具

一款专业的磁盘性能基准测试桌面应用程序，基于 PyQt6 构建，拥有精美的深色主题界面。

## 功能特性

- **顺序读写测试** - 测试磁盘顺序读写速度
- **随机读写测试** - 测试磁盘随机读写速度
- **混合负载测试** - 测试混合读写性能
- **块大小测试** - 不同块大小的性能对比
- **队列深度测试** - 不同队列深度的性能对比
- **历史记录** - 保存基准测试结果
- **结果对比** - 跨次运行结果对比
- **数据导出** - 支持导出为 CSV/TXT/HTML 格式

## 环境要求

- Python 3.10+
- Windows 10/11

## 安装

```bash
pip install PyQt6 psutil
```

## 运行

```bash
python main.py
```

## 界面主题

- 深色背景: #0a0a0a
- 卡片背景: #111122
- 渐变强调色: #667eea → #764ba2
