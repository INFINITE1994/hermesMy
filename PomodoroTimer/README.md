# PomodoroTimer - 番茄钟计时器

一个专业的番茄钟计时器桌面应用，基于 PyQt6 构建，帮助您提高工作效率。

## 功能特性

- 🍅 **番茄计时器** - 25分钟工作会话，5分钟短休息，15分钟长休息
- 📊 **统计面板** - 每日/每周/每月生产力统计
- ✅ **任务管理** - 将任务与番茄会话关联
- 🔔 **声音提醒** - 计时结束时的音频通知
- ⚙️ **自定义设置** - 调整工作/休息时长
- 🎯 **专注模式** - 工作期间阻止访问分散注意力的网站
- 🖥️ **系统托盘** - 最小化到系统托盘继续运行
- 🌙 **暗色主题** - 专业的深色界面设计

## 安装要求

- Python 3.10+
- Windows 10/11

## 安装步骤

```bash
# 克隆或下载项目
cd PomodoroTimer

# 创建虚拟环境 (推荐)
python -m venv venv
venv\Scripts\activate

# 安装依赖
pip install -e .

# 或直接安装依赖
pip install PyQt6
```

## 使用方法

```bash
# 运行应用
python main.py

# 或使用安装后的命令
pomodoro
```

## 界面预览

应用采用暗色主题设计：
- 背景色: #0a0a0a
- 卡片色: #111122
- 渐变强调色: #667eea → #764ba2

## 操作说明

1. **开始番茄** - 点击"开始"按钮启动25分钟工作计时器
2. **暂停/继续** - 随时暂停和继续计时
3. **跳过** - 跳过当前会话进入下一阶段
4. **添加任务** - 在任务面板中添加待办事项
5. **查看统计** - 切换到统计面板查看生产力数据
6. **设置** - 自定义计时时长和专注模式

## 专注模式

专注模式会在工作期间修改 hosts 文件，阻止访问常见的分散注意力网站：
- 社交媒体 (微博、抖音、B站等)
- 视频网站
- 游戏网站

⚠️ **注意**: 专注模式需要管理员权限来修改 hosts 文件

## 技术栈

- **GUI框架**: PyQt6
- **数据存储**: SQLite (本地存储)
- **音频**: PyQt6.QtMultimedia
- **打包**: PyInstaller (可选)

## 开发

```bash
# 安装开发依赖
pip install -e ".[dev]"

# 运行测试
pytest

# 打包为可执行文件
pyinstaller --onefile --windowed main.py
```
