# ImageBatch - 批量图片处理工具

<p align="center">
  <strong>🎨 专业级批量图片处理工具，支持多种图片操作</strong>
</p>

---

## ✨ 功能特性

### 📐 批量调整大小
- 按百分比缩放
- 按固定尺寸调整
- 保持宽高比选项

### 🔄 格式转换
- 支持 PNG、JPG、WebP、BMP 格式
- 批量一键转换

### 📦 批量压缩
- 可调节压缩质量 (1-100%)
- 实时预览压缩效果

### ✏️ 批量重命名
- 顺序命名 (001, 002, ...)
- 添加前缀/后缀
- 查找替换文件名

### 💧 添加水印
- 文字水印 (可调大小、颜色、位置、透明度)
- 图片水印

### 🔃 旋转翻转
- 顺时针/逆时针旋转 90°/180°/270°
- 水平/垂直翻转

### 👀 预览功能
- 缩略图网格预览
- 处理前后对比

---

## 🛠️ 安装要求

- Python 3.10+
- Windows 10/11

## 📥 安装步骤

```bash
# 克隆或下载项目
cd ImageBatch

# 安装依赖
pip install PyQt6 Pillow

# 运行程序
python main.py
```

---

## 🚀 使用方法

1. **添加图片**：点击"添加图片"按钮或直接拖拽图片到窗口
2. **选择功能**：点击左侧功能标签页
3. **配置参数**：设置处理参数
4. **预览**：点击"预览效果"查看处理结果
5. **执行**：点击"开始处理"批量处理图片
