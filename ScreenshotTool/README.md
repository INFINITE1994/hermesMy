# 📸 ScreenshotTool - 专业截图工具

一款功能强大的桌面截图工具，基于 PyQt6 构建，提供全屏截图、区域截图、窗口截图、标注、模糊处理、颜色拾取等专业功能。

---

## ✨ 功能特性

### 🖼️ 截图模式
| 功能 | 说明 | 快捷键 |
|------|------|--------|
| 🖥️ 全屏截图 | 一键捕获整个屏幕 | `Ctrl+Shift+F` |
| ✂️ 区域截图 | 鼠标拖拽选择任意区域 | `Ctrl+Shift+R` |
| 🪟 窗口截图 | 捕获指定窗口内容 | - |
| 📜 滚动截图 | 捕获滚动页面内容 | - |

### 🎨 标注工具
| 工具 | 说明 |
|------|------|
| ⬚ 矩形 | 绘制矩形标注框 |
| ○ 圆形 | 绘制圆形/椭圆标注 |
| → 箭头 | 绘制指向箭头 |
| ✏️ 画笔 | 自由手绘标注 |
| T 文字 | 添加文字标注 |
| 🌫 模糊 | 模糊/马赛克敏感区域 |
| 🎨 取色 | 从截图中拾取颜色 |

### 🛠️ 其他功能
- **💾 保存图片** - 支持 PNG、JPEG、BMP 格式
- **📋 复制到剪贴板** - 一键复制截图
- **📂 截图历史** - 自动保存，带缩略图预览
- **↩️ 撤销** - 支持撤销标注操作
- **⚙️ 样式设置** - 可调颜色、线宽、字号

## 🚀 安装与运行

### 系统要求
- Python 3.10+
- Windows 10/11

### 安装依赖

```bash
# 使用 pip 安装
pip install PyQt6 Pillow pynput

# 或使用项目配置安装
pip install -e .
```

### 运行程序

```bash
python main.py
```

## 📝 使用说明

### 区域截图
1. 点击「区域截图」按钮或按 `Ctrl+Shift+R`
2. 屏幕变暗后，用鼠标拖拽选择要截取的区域
3. 松开鼠标完成截图
4. 按 `ESC` 取消

### 标注图片
1. 截图后自动进入编辑模式
2. 在左侧面板选择标注工具
3. 调整颜色、线宽、字号
4. 在图片上拖拽绘制标注
5. 点击「撤销」可删除最后一个标注

### 模糊处理
1. 选择「模糊」工具
2. 在需要模糊的区域拖拽
3. 敏感信息将被自动模糊处理

### 取色器
1. 选择「取色」工具
2. 在截图上点击要取色的位置
3. 当前颜色会自动更新为拾取的颜色
